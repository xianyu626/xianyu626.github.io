NOIP模拟赛 10.25

NOIP模拟赛 10.25

这场三题接近爆零…好惨啊，考虑单独开一篇。

树

时间限制：0.5s空间限制：128MB

Description

$cwystc$ 是天才学院最优秀的学生。在成功 AK $NOI2018$ 之 后，$cwystc$ 开始思考他智商为什么这么高，他翻出了族谱，开始探究生命的本源。

家谱可以看做一棵以 $1$ 为根的树，第 $i$ 个人的智商为 $w_i$ 。 $cwystc$ 会进行 $q$ 次探究，对于第 $i$ 次探究，$cwystc$ 初始智商为 $c_i$ ，他将从第 $u_i$ 个人开始向 $1$ 方向探索，直到第 $v_i$ 个人。对于路径上某个人 $k$ ，若 $k$ 的智商严格高于 $cwystc$ 的智商，$cwystc$ 会努力学习以达到与 $k$ 相等的智商。 $cwystc$ 想知道对于每次探究，他需要努力学习多少次。

Input

第一行输入两个整数 $n , q$ ,表示节点数和询问数。

第二行输入 $n$ 个整数 ,表⽰第 $i$ 个点的智商。

第三行至第 $n+1$ 行每行输入两个数 $x,y$ ,表示树上一条边。

第 $n+2$ 行至第 $n+q+1$ 行每行三个数 $u,v,c$ 表示一次探究。（保证 $v$ 是 $u$ 的 祖 先）

Output

输出 $q$ 行,每行一个数表示探究过程中 $cwystc$ 需要努力学习的次数

Sample Input

5 4
3 5 1 2 4
1 2
1 3
2 4
3 5
4 2 1
4 2 2
4 2 3
5 1 5

Sample Output

2
1
1
0

数据范围

对于 $10\%$ 的数据: $n\le 1000$

对于另外 $30\%$ 的数据:家谱树为一条链

对于 $100\%$ 的数据: $n,q,w_i,c\le100000$

Solution

这道题的实质时要求出一棵树上一条 $u$ 到 $v$ 的链上以 $c$ 为第一个数的单调上升序列的长度，且保证 $v$ 为 $u$ 的祖先

先用倍增维护出：$u$ 到 $root$ 的路径上第一个 $w_i$ 比 $val$ 大的点。这个可以通过倍增维护最大值， $log~ n$ 求出。然后利用这个，处理出每个点的祖先中第一个比其 $w_i$ 大的点 $fa_i$，由于每个点都有一个唯一的 $fa_i$（当然有的点 $fa_i=0$），这就构成了一颗新树。

对于每一个询问，我们先求出链上第一个比 $c$ 大的点 $u$ ，再通过新树倍增求出 $u$ 到 $v$ 这条路径的长度就好了。

环

时间限制：1s空间限制：128MB

Description

$cwystc$ 是天才学院最优秀的学生。在成功保送之后，开始每天陪妹子散步。天才学院共有 $n$ 个景点，每两个景点之间有且仅有一条有向边相连，但是 $cwystc$ 只确定了其中的 $e$ 条边。

妹子希望每天的路线都不相同，而且她还希望 $cwystc$ 能够送她回家 (即回到路径的起点) ，但是 $cwystc$ 很懒，他每天只希望走最短的路线。他想知道他能陪伴妹子的期望天数。

如果天才学院中不存在满足的路线，那我们会认为 $cwystc$ 在该方案中陪伴妹子的天数为 $0$ 。

Input

第一行读入两个整数 $n,e$ 表示节点数及 $cwystc$ 已确定的有向边边数。

接下来 $e$ 行，每行两个整数 $x,y$ 描述 $cwystc$ 确定的边。

Output

输出一个整数表示期望陪伴妹子的天数。

Sample Input

4 2
1 2
1 3

Sample Output

750000006

数据范围

对于 $30\%$ 的数据: $n\le 300$
对于另外 $20\%$ 的数据: $e=0$
对于另外 $20\%$ 的数据: $e=\frac{n\times n-1}{2}$
对于 $100\%$ 的数据: $n\le 100000,e\le 1000000$

Solution

首先这个图是一个竞赛图，关于竞赛图我们有一个结论：竞赛图要么无环，否则必有三元环。（这个可自行百度）所以问题就转化成了求三元环期望个数。

考虑如果每条边都不确定（即两个方向概率相等），个数期望是多少：对于任意三个点，都有 $2^3$ 种连边可能，其中 $2$ 种会形成三元环，所以此时的个数期望就是 $ \frac{1}{4} · C^3_n$ 。

接下来考虑每条确定的边会对答案造成什么影响，先考虑任意三个点会有哪些情况，如下图（这些贡献是在每三个点的组合贡献为 $\frac{1}{4}$ 的基础上）

冷静分析，发现若只有一条边确定，就没有影响，两条边确定根据其是否同向分两种情况，三条边确定时的贡献其实已经在两条边中被算过了，不需要再算，所以实际上我们只需要计算两条边的两种情况各有多少种就好了。

该怎么计算？对每个点处理出其出度和入度记为 $x,y$ ，发现出点和入点中各任选一个点即可组成边异向的情况，出点或入点中任选两个点组成边同向的情况

因此，每个点对答案的贡献为 $+\frac{xy}{4} - \frac{1}{4}C^2_x - \frac{1}{4}C^2_y $

礼物

时间限制：2s空间限制：512MB

Description

$cwystc$ 是天才学院最优秀的学生。在成为人生赢家之后，$cwystc$ 开始陪妹子逛商店。

商店中总共 $n$ 个礼物，对于第 $n$ 个礼物，包含 $a_i$ 个红宝石和 $b_i$ 个绿宝石，$cwystc$ 和妹子想各买一个礼物，然后将其中的宝石串成一条链，$cwystc$ 想知道对于每一种购买方案的排放方案数的和。

由于方案数过大，所以输出对 $1000,000,007$ 取模。

一句话题意：求 $\sum \limits_{i!=j} \binom{a_i+b_i+a_j+b_j}{a_i+a_j}$

Input

第一行输入一个正整数 $n$ 。

第二行到第 $n+1$ 行每行两个正整数 $a_i$ 和 $b_i$ 表示第 $i$ 个礼物中包含 $a_i$ 个红宝石和 $b_i$ 个绿宝石。

Output

输出一个整数表示方案数。

Sample Input

3
1 1
1 1
1 2

Sample Output

52

数据范围

对于 $20\%$ 的数据: $n\le 5000, a_i,b_i\le 1000000$

对于另外 $30\%$ 的数据: $a_i,b_i\le 4000$

对于另外 $20\%$ 的数据: $n\le 50000,\Sigma a_i+\Sigma b_i\le 2000000$

对于 $100\%$ 的数据: $n\le 100000,\Sigma a_i+\Sigma b_i\le 20000000$
