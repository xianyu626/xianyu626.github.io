NOIP2018赛前集训（一周倒计时）

NOIP赛(tui)前(yi)倒计时：7 Days

11.05

Merchant

时间限制：1s空间限制：512MB

Description

有 $n$ 个物品，第 $i$ 个物品有两个属性 $k_i, b_i$ ，表示它在时刻 $x$ 的价值为 $xk_i+b_i$ 。

当前处于时刻 $0$ ，你可以选择不超过 $m$ 个物品，使得存在某个整数时刻 $t$ , $t ≥ 0$ ，你选择的所有物品的总价值大于等于 $S$ 。

给出 $S$ ，求 $t$ 的最小值。

Input

第一行三个整数 $n, m, S$ 。

接下来 $n$ 行，第 $i$ 行两个整数 $k_i, b_i$ 。

Output

一行一个整数表示答案。

Sample Input

3 2 100
3 9
-2 50
4 1
3 2 100
-1 49
-2 50
1 -998244353

Sample Output

13
998244453

数据范围

对于所有数据，有 $1\le m \le n \le 10^6, −10^9 \le b_i \le 10^9, −10^6 \le k_i \le 10^6, 0 \le S \le 10^{18}$ 。

数据保证有解，且答案不超过 $10^9$ 。

  • Subtask1($22\%$), $n \le 20$ 。
  • Subtask2($36\%$), $n \le 10^5,0\le k_i\le 10^4$ 。
  • Subtask3($8\%$), $k_i \le 0$ 。
  • Subtask4($12\%$), $n \le 10^5$
  • Subtask5($22\%$), 没有特殊的约束。

Solution

观察一下，发现物品实质是个一次函数，题目求取其中至多 $m$ 个一次函数，满足和大于 $S$ ， 此时 $x$ 的最小值，由于对于每一个 $x$ 值，我们取得是所有和中的最大值，若把这个最大值与 $x$ 的关系看成一个函数，会发现其先单调递减，再单调递增，因此先判 $x=0$ 时是否满足条件，若不满足，则单调递减的部分一定都不满足，可以看作一个单调递增函数，直接二分答案即可。

具体实现时，要用到一种基于快速排序思想的 $O(n)$ 求第 $k$ 大的算法，这样整体复杂度就是$O(n\log n)$的

实际程序中用了STL里的 nth_element 作用是将第 $n$ 小的数放在第 $n$ 位并把比它小的数放它前面，把比它大的数放它后面。

关于单调性的感性证明：若函数开始单调递增，此时所取的一次函数集合为 $A$ ，当 $x$ 增大时若任意一次函数 $a\in A，a(x)< 0$ 会使函数值变小，不再选这个一次函数一定不会使函数值变小

Code

Equation

时间限制：1.5s空间限制：512MB

Description

有一棵 $n$ 个点的以 $1$ 为根的树, 以及 $n$ 个整数变量 。树上 的父亲是 ，每条边 有一个权值 ，表示一个方程 ，这 个方程构成了一个方程组。

现在给出 $q$ 个操作，有两种类型:

  • $1~u~v~s$ ，表示询问加上 $x_u+x_v=s$这个方程后，整个方程组的解的情况。
    具体来说，如果方程有唯一解，输出此时 $x_1$ 的值；如果有无限多个解，输出inf；如果无解，输出none。 注意每个询问是独立的。
  • $2~u~w$ ，表示将 $w_u$ 修改为 $w$ 。

Input

第一行两个整数 $n,q$ 。

接下来 $n−1$ 行，第 $i$ 行有两个整数 $f_i+1$ 和 $w_i+1$ 。

接下来 $q$ 行，每行表示一个操作，格式见问题描述。

Output

对于每个询问输出一行表示答案。

Sample Input

2 7
1 4
1 1 2 5
1 1 2 4
1 1 1 3
1 2 2 6
2 2 3
1 2 2 10
1 2 2 -10

Sample Output

none
inf
none
1
-2
8

数据范围

对于所有数据，有 $1\le n,q\le 10^6,1\le f_i\le i−1,1\le u,v\le n,−10^3\le w,w_i\le 10^3,−10^9\le s\le 10^9$ 。

  • Subtask1($3\%$), $n\le 10,q=0$ 。
  • Subtask2($18\%$), $n=2$ 。
  • Subtask3($32\%$),$n,q\le 10^3$
  • Subtask4($33\%$),$n,q\le 10^5$
  • Subtask5($14\%$),没有特殊的约束。

Solution

模拟一下，发现树上每一个点根据其深度的奇偶性都可以表示成 $x_i=x_1+k$ 或 $x_i=-x_1+k$ 的形式。

对于一个询问 $x_u+x_v=s$ ，两式相加，若两点深度奇偶性不同，则可得 $x_u+x_v=t$ ，此时判断是否 $s=t$ ，输出inf或none，若奇偶性相同则可得 $x_u+x_v=t+2x_1$ 或 $x_u+x_v=t-2x_1$，由此解出 $x_1$ ，注意判一下其是否为整数。

对于一个修改，发现实际上它影响的是它的子树，运用$dfs$序将其转化为区间修改，单点查询的问题，用树状数组维护。

整体复杂度 $O((n+q)\log n)$
